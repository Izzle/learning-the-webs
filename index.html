The Box Model

Most HTML elements can be thought of as boxes. Recall that CSS is based on the box model of the DOM.

  • padding is the space around the content
  • border is the line that sits just outside the padding
  • margin is the space around the outside of the element ('padding' around the border)

Git basics

After you've initialized your git repository, its pretty simple to add and save changes.

Git Add

The git add command adds changes you've made in the current working directory to a staging area. You can update existing files in the directory or add new files entirely. None of these changes are permanent until the next time you commit them.

git add <file>

Stages all changes to <file> to be added in the next commit.

git add *

Stages all changes in the current directory for the next commit. Be careful before using this catch-all statement.

git add -p

Starts an interactive staging session that prompts you to decide whether or not to keep a changes one chunk at a time.

Git Commit

The git commit command saves all the pending changes in the staging area to a new verison of the project. Having multiple verisons allows you to experiment freely without worrying about doing irreparable damage.

git commit

This will commit the changes you've made to a new snapshot. It will also open a text editor so you can add comments detailing the work you've done. Always comment the work you do - you will thank yourself later.

git commit -m "<comment>"

Commits the changes you've made, but saves your <comment> instead of opening a text editor

git commit -a

Commits all changes in the directory.

Git Push

After you've made your final commits locally, you should push your changes to the branch you are working on. On a community project you'd make a pull request, but on a repo that you own (such as this website) you can simply push directly to the branch with a...

git push

After pushing your commit, you'll be asked for your passphrase - just enter the passphrase and thats it! Once you've pushed the commit to somewhere like github the latest version of your project will be safe in the cloud!

Color

You can add color in CSS in many different ways. You can just use a color's name.

color: black;

Color can also be expressed with hex code. Just as the Decimal system is a base-10 number system (ranging from 0-9) - Hexidecimal is a base-16 number system that uses letters to denote the last 6 digits (A=11, B=12.. F=16); you can read more about hexidecimal here. Hex uses an RGB color system of 6 digits to represent color where each pair represents a color. For example, 0 represents the complete absence of color (black) and F represents the purest form of a color (red, green, or blue).

#000000

Another way you can represent color in CSS is with RGB values. The RGB for black is

rgb(0, 0, 0)

And the RGB value for white is

rgb(255, 255, 255)

Instead of using 6 hexidecimal digits like you did with hex color, when using RGB you specify each color's brightness with a number ranging from 0 to 255.

When you used hexidecimal for color, we used two base-16 numbers for each color. Doing the math, 16 times 16 gives us 256 different variations. Using RGB, which starts counting from zero, we have the same number of possible values for color.
